BANNER IMAGES GUIDE
===================

Please add your banner images to this folder with the following names:

HERO BANNERS (Recommended size: 1920x800 pixels):
------------------------------------------------
1. hero-banner-1.jpg - First hero slide (New Season / Discover Your Style)
2. hero-banner-2.jpg - Second hero slide (Elegant Sarees)
3. hero-banner-3.jpg - Third hero slide (Modern Dresses)

PROMOTIONAL BANNER (Recommended size: 1920x600 pixels):
-------------------------------------------------------
4. promo-banner.jpg - Promotional/Sale banner section

IMAGE TIPS:
-----------
- Use high-quality JPEG images for best performance
- Optimize images for web (compress to reduce file size)
- Ensure images are at least 1920px wide for full-width display
- Use fashion/clothing related images that match your brand
- Dark or muted background images work best with white text overlay

You can download images from the FMLK website or use your own product photos.
